# AI-Powered Debt Relief & Financial Recovery Platform

## Overview

The AI-Powered Debt Relief & Financial Recovery Platform is a full-stack web application designed to help users manage debt, analyze their financial health, predict settlement offers, and generate AI-powered debt negotiation letters.

The platform uses React for the frontend, FastAPI for the backend, SQLite for data storage, and Google Gemini AI for intelligent financial assistance.

---

## Features

- User Registration & Login
- Secure JWT Authentication
- Financial Profile Management
- Loan Management
- AI-Based Debt Settlement Prediction
- AI Negotiation Letter Generation
- Financial Dashboard
- History & Activity Logs
- Responsive User Interface

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python run.py
```

Backend URL:

```
http://127.0.0.1:8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend URL:

```
http://localhost:5173
```

## Application Workflow

1. Register a new user account.
2. Login securely using JWT authentication.
3. Complete the financial profile.
4. Add loan details.
5. Analyze debt settlement predictions.
6. Generate an AI-powered negotiation letter.
7. Download or copy the generated proposal.
8. View history and previous negotiation letters.
